The **Attio Workflows Handbook** documents the new Workflows engine: every block, every credit cost, and follow-along go-to-market (GTM) builds you can ship. Attio shipped that engine in June 2026. It added AI agent blocks, custom JavaScript, and Model Context Protocol (MCP) tool access. It also added support for more than one trigger per workflow, and a cheaper credit model. The official docs explain what each block does, but they do not show you how to build with them. This handbook fills that gap. It is also open source, so if you spot a mistake, [open a pull request](https://github.com/80x-djh/attio-workflows-handbook).
