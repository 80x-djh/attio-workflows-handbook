## Why this work is hard

Attio Workflows sit where CRM design, go-to-market (GTM) operations, APIs, AI agents, and data hygiene all meet. Clicking the blocks is rarely the hard part. The hard part is the judgment behind them:

- Which [trigger](/explanation/triggers) should start the workflow.
- Where the source of truth for the data lives.
- Which [path should stay free](/reference/credits-and-pricing) so you do not waste credits.
- Which write can cause an [infinite loop](/explanation/infinite-loops-and-safety), where a workflow keeps triggering itself.
- Which records should never reach an AI step.

These offers cover that work.
