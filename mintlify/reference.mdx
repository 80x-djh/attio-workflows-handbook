Attio Workflows has 11 triggers and 33 steps. Each block has its own reference page with inputs, outputs, credit cost, and a worked example. Every workflow you build is assembled from these blocks. Each reference page follows the same fixed layout, so once you have read one you can skim the rest.

Credit costs follow one rule. Triggers, logic, and lookups are free. Writes and external calls cost 1 credit, and AI is priced by the tokens it uses. The [Credits & pricing explained](/reference/credits-and-pricing) page carries the full table and the per-plan allowances. Every block page links to it rather than repeating it.

## Agents

An agent block runs an AI agent against your workspace. Its tool access stays read-only, gated by the "Run as" permission model. That means the agent can read your data but cannot change it.

<CardGroup cols={2}>
  <Card title="Custom agent" icon="bot" href="/reference/agents/custom-agent">
    Runs a configured AI agent with read-only tools over your data.
  </Card>
  <Card title="Web agent" icon="globe" href="/reference/agents/web-agent">
    Runs an AI agent that researches the public web.
  </Card>
</CardGroup>

An AI block applies a model to a record or to text. Each one is priced by the tokens it uses, not at a flat rate.

## Records

A record block reads or writes the records in an object. Each write costs 1 credit. Reads are free.

<CardGroup cols={2}>
  <Card title="Create record" icon="square-plus" href="/reference/records/create-record">
    Creates a new record in an object.
  </Card>
  <Card title="Create or update record" icon="copy-plus" href="/reference/records/create-or-update-record">
    Upserts a record, matching on an attribute you choose.
  </Card>
  <Card title="Update record" icon="square-pen" href="/reference/records/update-record">
    Sets attribute values on an existing record.
  </Card>
  <Card title="Find records" icon="search" href="/reference/records/find-records">
    Looks up records that match a filter, with no credit cost.
  </Card>
</CardGroup>

<CardGroup cols={2}>

  <Card title="Switch" icon="split" href="/reference/conditions/switch">
