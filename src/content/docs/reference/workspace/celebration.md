**Celebration** displays an in-app animation.

| | |
| --- | --- |
| **Type** | Workspace step |
| **Credit cost** | Free |

## Inputs

| Input | Required | Notes |
| --- | --- | --- |
| Type | Yes | Ceiling drop, Fireworks, or Cannons. |
| Target | Yes | Users who should see it. |

## Gotchas

- Use sparingly. It is for meaningful moments, not every stage change.
- In legacy workflows, Confetti maps to Ceiling drop.

## Example

Deal marked Won -> Celebration for owner and manager -> Slack alert.
